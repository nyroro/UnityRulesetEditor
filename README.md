UnityRulesetEditor
=======

UnityRulesetEditor is a tool to edit rulesets for [Roslyn analyzer in Unity](https://docs.unity3d.com/Manual/analyzer-scope-and-diagnostics.html).
Window -> Rulesets to open the window
